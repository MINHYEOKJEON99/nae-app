# NAE App - 프론트엔드 개발 계획

## Context

IT 뉴스 브리핑 앱. 바쁜 개발자가 트렌드를 빠르게 확인할 수 있는 토스 스타일 모바일 퍼스트 앱.

**현재 상태**: Next.js 16.1.6 + React 19 + TypeScript 프로젝트 생성됨, 보일러플레이트 삭제 상태

**핵심 결정:**

- 프론트엔드 먼저 (목업 데이터 → 백엔드 연결)
- Emotion 스타일링
- 하단 탭 바 네비게이션
- 토스 앱 디자인 (둥근 카드, 넉넉한 여백, pill 버튼, 배경색 분리)

| 결정 | 선택 | 이유 |
|------|------|------|
| Emotion SSR | 수동 레지스트리 (SWC 플러그인 X) | App Router 서버 컴포넌트 호환 |
